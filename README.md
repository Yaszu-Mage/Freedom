This is a Minecraft Plugin meant for Paper that depends on:
Requires Better Model
Requires Packet Events
Requires World Edit
Requires Skins restorer
This mod focuses on adding several movesets based on colors, along with custom dimensions, items, recipies etc. 
The movesets include
Purple,
Green,
Red,
Blue,
Orange,
Black,
Yellow,
Mocha,
Cafe,
Astral
Arcanus
and a couple more!
There is a miriad of different recipes for you to find! 
(Admins use /bestow to give yourself items)
There is a /sudo command to seperate admin profiles HARDCODED, you will be deoped on restart.
